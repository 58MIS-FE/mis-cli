## MIS-FE-CLI 

#### A fast development template

```
   npm install mis-cli -g 

``` 

#### Commands 

```
  mis init
  mis init -f [--force]' 如果目标目录存在，强制覆盖
  mis init -c [--clone]' 在获取远程预置时使用Git克隆
```
